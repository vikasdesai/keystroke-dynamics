Keystroke Dynamics Demo

Runs in a browser

This project shows how a website could authenticate / identify a user based on how the user uses her keyboard. The project uses ml.js a machine learning toolkit written in Javascript. This project was initially designed for a presentation for the 2018 APJ RSA Conference.

Click on Keystroke Dynamics Demo to load the demo. It is a self contained demo which will run in your browser.
